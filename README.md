# Marp to Google Slides (GAS)

Markdown (Marp形式)で記載された構成テキストを、Googleスライドの「指定テンプレート」にデザインを崩さずに一括で流し込むための専用Google Apps Script（GAS）ツールです。

## 特徴
1. **Googleスライドのコーポレートデザイン（テンプレート）を完全保持**
   * MarpからPPTXを経由してGoogleスライドにインポートすると、マスターレイアウトが破壊される問題を解決します。
2. **スピーカーノート（台本）への完全対応**
   * Markdown内の `<!-- speaker_note -->` 以下を自動認識し、スライドのノート領域へ流し込みます。
3. **安全なサイドバーUIによる流し込み**
   * GASのコードエディタに直接Markdownを貼り付けるのではなく、スライドのメニュー画面から呼び出せる専用UI（サイドバー）を提供し、改行コードやバッククォートの構文エラーを完璧に回避します。

## 導入方法
1. 流し込み先の「空のGoogleスライド」を開きます。
2. 上部メニューから **「拡張機能」>「Apps Script」** を開きます。
3. デフォルトで書かれている `function myFunction() {}` などをすべて消去し、本リポジトリの `Code.gs` の内容をすべて貼り付けて保存（Ctrl+S）します。
4. 元のGoogleスライド画面に戻り、F5キー等で画面をリロード（再読み込み）します。

## 使い方
1. リロード後、上部のメニューバー右端に **「📘 Markdown流し込み」** という独自のメニューが出現するのでクリックします。
2. **「スライド自動生成ツールを開く」** を選択します。
3. 画面右側にサイドバーが出現するので、そこへMarp形式のMarkdownテキストを貼り付けます。
4. **「自動生成する」** ボタンを押すと、数秒〜十数秒で全スライドが生成されます。
